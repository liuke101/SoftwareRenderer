# SoftwareRenderer

一个简易的软件光栅渲染器。




打开项目：

从 github 下下来整个课程的文件，打开 **visual studio**--> **继续但无需代码** --> **文件** --> **新建** --> **从现有代码创建项目** --> 项目类型选择 **Visual C++**--> 项目文件位置选择**对应课时的文件夹**，项目名称随意，点击下一步 --> 项目类型选择**控制台应用程序**，点击完成。
